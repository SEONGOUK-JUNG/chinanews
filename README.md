# 차이나뉴스 · CHINANEWS.KR

중국 경제·원자재·정책을 한눈에 파악하는 인텔리전스 플랫폼.

**🌐 Live:** [https://chinanews.kr](https://chinanews.kr)

## 개요

차이나뉴스(chinanews.kr)는 전세계 무역·제조·ESG 의사결정자를 위한 중국 시장 모니터링 서비스입니다. 희토류부터 폴리에틸렌, 식량까지 실시간 원자재 가격과 정책 동향을 한국어·영어·중국어·일본어로 제공합니다.

## 주요 기능

- **실시간 원자재 가격** — 중국 SunSirs 기반 17개 핵심 품목 일일 추적
- **중국 경제 뉴스** — 관영·산업 매체 통합 큐레이션
- **환율·거시지표** — CNY/KRW·USD·EUR·JPY 실시간
- **정책·규제 타임라인** — 한·중·글로벌 3축 정책 추적

## 기술 스택

- **프론트엔드:** 정적 HTML/CSS/JavaScript (단일 파일 SPA)
- **차트:** Chart.js
- **호스팅:** GitHub Pages
- **도메인:** chinanews.kr (후이즈)
- **데이터 파이프라인:** Python 크롤러 (별도 리포)
- **에디토리얼 파트너:** 브릿지그룹 (㈜에이브릿지 · ㈜에코브릿지)

본 매체는 다음 6개 편집 원칙을 준수합니다:
