# 차이나뉴스 터미널 · CHINANEWS Terminal

> 한국인을 위한 중국 주식·원자재 금융 단말기

**🌐 Live:** https://chinanews.kr

---

## 개요

차이나뉴스 터미널(chinanews.kr)은 한국인 투자자가 **중국 본토(A주)·홍콩(H주)·ADR 중국 주식**과 **원자재·환율·금리**를 한 화면에서 모니터링할 수 있는 블룸버그 스타일 금융 단말기입니다.

한국 주식과 DART 공시도 함께 지원합니다.

## 주요 기능

- **중국 주식 실시간 시세** — A주(상해·심천), H주(홍콩), ADR(NYSE·NASDAQ)
- **한국 주식 지원** — KOSPI·KOSDAQ + DART 공시
- **원자재 데이터** — SunSirs 기반 17개 핵심 품목 일일 추적
- **환율·금리** — CNY/KRW·USD/CNY·USD/KRW·10Y 국채 수익률
- **ETF·옵션** — 중국 ETF, 홍콩 옵션, ADR 옵션
- **공시 통합** — DART·CSRC·SEC EDGAR
- **AI 트레이드 어시스턴트** — Groq 기반

## 투명성 원칙

본 단말기는 **데이터가 없는 위치에 임의의 숫자를 채워 넣지 않습니다.**

- `REAL` — 실측 데이터
- `API 필요` — 외부 API 연결 시 채워짐
- `PENDING` — 다음 페이즈 연동 예정
- `—` — 데이터 없음
